Angular Material - Typography

General Styling

By default, Angular Material doesn't apply any global CSS. In order to make use of Material Typography, you need to specify the appropriate style classes.

Available Typography Levels

  • display-4, display-3, display-2, and display-1 - Large, one-off headers, usually at the top of the page (e.g. a hero header).
  • headline - Section heading corresponding to the <h1> tag.
  • title - Section heading corresponding to the <h2> tag.
  • subheading-2 - Section heading corresponding to the <h3> tag.
  • subheading-1 - Section heading corresponding to the <h4> tag.
  • body-1 - Base body text.
  • body-2 - Bolder body text.
  • caption - Smaller body and hint text.
  • button - Buttons and anchors.

Example

<p class="mat-caption">Material Design Caption</p>

Typography Sections

Alternatively, you can define a markup container with the mat-typography class.

Example

<section class="mat-typography">
    <h1>Styled h1 Header</h1>
</section>

Continuing from the previous walkthroughs, in ClientApp/app/components/app/app.component.html, add the mat-typography class to the md-sidenav-container element: 

<md-sidenav-container [ngClass]="materialTheme" class="mat-typography">

This ensures that any of the markup contained in the web app receives the appropriate font.

Self-hosting Fonts

By default, the themes in Angular Material use the Roboto font-face for the overall application typography. Also, the Material Design Icons are used for icon components.

In order to integrate these font-faces into the build, the following steps need to be taken:

  1. Install roboto-fontface and webpack-material-design-icons modules as devDependencies: 

    npm install --save-dev roboto-fontface webpack-material-design-icons
  2. Add roboto-fontface/css/roboto/roboto-fontface.css and webpack-material-design-icons to the vendors array in webpack.config.vendor.js.
  3. Run the npm run build script to install the dependencies and generate the build files.

Now, the web app self-hosts the fonts and icons used by Angular Material. To demonstrate, add an icon to the Menu button in ClientApp/app/components/app/app.component.html:

ClientApp/app/components/app/app.component.html 
<button md-button (click)="sidenav.open()"><md-icon>menu</md-icon> Menu</button>
